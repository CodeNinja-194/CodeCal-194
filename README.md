CodeCal is a premium coding contest tracker designed for competitive programmers. Stay ahead of the curve with real-time updates on contests from all major platforms including LeetCode, CodeChef, GeeksforGeeks, Codeforces, and AtCoder.

## ‚ú® Features

- **Dynamic Contest Discovery**: View LIVE, Upcoming, and Today's contests at a glance.
- **Interactive Calendar**: Hover over calendar events to see contest details and jump directly to the contest page.
- **Premium Aesthetics**: Liquid gradients, glassmorphism, and smooth micro-animations.
- **Dual Theme Support**: Seamlessly switch between a sleek Dark Mode and a crisp Light Mode.
- **Smart Filters**: Filter contests by platform to focus on what matters to you.
- **Alert System**: Enable notifications to never miss a contest again.
- **DSA Roadmap**: In-built access to a curated DSA Learning Roadmap.

## üöÄ Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Codeninja-194/CodeCal.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## üì¶ Deployment

### Deploying to Vercel

To deploy your project to Vercel using the CLI, follow these steps:

1. **Install Vercel CLI** (if not already installed):
   ```bash
   npm i -g vercel
   ```

2. **Login to Vercel**:
   ```bash
   vercel login
   ```

3. **Deploy Preview**:
   ```bash
   vercel
   ```

4. **Deploy to Production**:
   ```bash
   vercel --prod
   ```
